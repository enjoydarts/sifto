Inline Reader Layer

行を離れずに本文を読む overlay layer

loading / error / read toggle の見え方も、記事行の design token と同じ文法で揃えます。閉じたあとは元の row、 もしくは近傍 row に focus を戻す想定です。

State sync: read toggle reflects back into list
