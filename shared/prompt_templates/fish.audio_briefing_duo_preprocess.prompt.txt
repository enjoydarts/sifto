## 【トーン方針】
- 丁寧だが堅すぎない
- 親しみはあるが、崩しすぎない
- 感情は軽く乗せる程度（演技過多はNG）
- “説明がうまい人が普通に話している感じ”で、会話らしいリズムと軽い抑揚をつける
- 相槌や聞き返し、軽い笑い（くすくす程度）を自然に入れて会話感を出す

## 【基本ルール】
- タグは必ず `[ ]` で囲む
- 各文の先頭に1つだけ付ける（基本は短め）
- タグは控えめだが、会話の流れに合わせてわずかにバリエーションを持たせる
- 同じタグの連続は避ける
- `<|speaker:0|>` と `<|speaker:1|>` の話者タグは絶対に消さない・入れ替えない
- 笑い声は軽く自然に（くすくす・軽く笑いながら程度）

## 【文頭タグ（これをベースに自然にバリエーションを生成してください）】
軸は以下：
- `[落ち着いて]`
- `[やさしい声]`
- `[自然に]`
- `[明るく]`
- `[柔らかく]`
- `[ゆっくりめに]`
- `[強調]`
- `[淡々と]`
- `[相槌を打つように]`
- `[少し聞き返すように]`
- `[興奮して]`
- `[クスクス笑い]`

会話らしさを出すために適宜混ぜて：
- `[軽く笑いながら]`
- `[クスクス笑い]`
- `[穏やかに]`
- `[軽やかに]`
- `[丁寧に少し温かく]`
- `[リズムよく]`

基本は「落ち着いて / 自然に」を軸に、説明の流れや掛け合いに合わせて少し変化させる。

## 【日本語最適化】
### ■ 外来語・英語の扱い
- 必ず自然な日本語カタカナ表記に変換（例: API → エーピーアイ、Cloud → クラウド、Ruby → ルビー、PostgreSQL → ポストグレスキューエル）
- 強引で不自然な変換は避ける
- 全文をカタカナにするのは禁止

### ■ 数字・小数
- 「23.5」→ 「二十三点五」のように自然な読みに変換
- 必要に応じて `[自然な読みで]` を追加

### ■ 読み上げ正規化
- 通貨:
  - `$42.50` → `四十二ドル五十セント`
  - `£1,001.32` → `千一ポンド三十二ペンス`
- 数値:
  - `1234` → `千二百三十四`
  - `3.14` → `三点一四`
  - `3.5` → `三点五`
  - `100%` → `百パーセント`
  - `100km` → `百キロメートル`
- 電話番号や連番:
  - `555-555-5555` → `五五五、五五五、五五五五`
- 序数・ローマ数字:
  - `2nd` → `二番目`
  - `XIV` → `十四`
  - ただし称号や固有名詞の一部なら文脈優先
- 分数:
  - `⅔` → `三分の二`
- 略語:
  - `Dr.` → `ドクター`
  - `Ave.` → `アベニュー`
  - `St.` → `ストリート`
  - ただし `St. Patrick` のような固有名詞は文脈優先
- 操作キー:
  - `Ctrl + Z` → `コントロールゼット`
- URL・パス:
  - `elevenlabs.io/docs` → `イレブンラブズ ドット アイオー スラッシュ ドックス`
- 日付:
  - `2024-01-01` → `二千二十四年一月一日`
  - `01/02/2023` → 文脈と地域に応じて自然な日付読みにする
- 時刻:
  - `14:30` → `十四時三十分`
- 住所:
  - 英語住所は、無理に翻訳せず、日本語として読める自然な形に展開する
