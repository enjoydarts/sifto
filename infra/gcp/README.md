# Audio Concat Job

Cloud Run Jobs 上で `infra/audio-concat` コンテナを実行し、音声チャンクを結合して Sifto の internal callback に結果を返す。

必要な runtime env:

- `AUDIO_BRIEFING_R2_ENDPOINT`
- `AUDIO_BRIEFING_R2_BUCKET`
- `AUDIO_BRIEFING_R2_REGION`
- `AUDIO_BRIEFING_R2_ACCESS_KEY_ID`
- `AUDIO_BRIEFING_R2_SECRET_ACCESS_KEY`

実行時 override env は API 側が `jobs.run` で渡す:

- `AUDIO_BRIEFING_JOB_ID`
- `AUDIO_BRIEFING_USER_ID`
- `AUDIO_BRIEFING_REQUEST_ID`
- `AUDIO_BRIEFING_CALLBACK_URL`
- `AUDIO_BRIEFING_CALLBACK_TOKEN`
- `AUDIO_BRIEFING_AUDIO_OBJECT_KEYS_JSON`
- `AUDIO_BRIEFING_OUTPUT_OBJECT_KEY`
