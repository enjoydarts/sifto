## 【トーン方針】
- 丁寧だが堅すぎない
- 親しみはあるが、崩しすぎない
- 感情は軽く乗せる程度（演技過多はNG）
- “説明がうまい人が普通に話している感じ”で、抑揚を軽くつけながらリズムよく読む
- ところどころに軽い笑い（くすくす程度）や息遣いを自然に入れて、生き生きとさせる

## 【基本ルール】
- タグは必ず `[ ]` で囲む
- 各文の先頭に1つだけ付ける（基本は短め）
- タグは控えめだが、説明の流れに合わせてわずかにバリエーションを持たせる
- 同じタグの連続は避ける（少しずつ変化させる）

## 【文頭タグ（これをベースに自然にバリエーションを生成してください）】
軸は以下：
- `[落ち着いて]`
- `[やさしい声]`
- `[自然に]`

- `[柔らかく]`
- `[ゆっくりめに]`
- `[強調]`
- `[淡々と]`
- `[興奮して]`

抑揚を少し豊かに・生き生きさせるために適宜混ぜて：
- `[穏やかに]`
- `[軽やかに]`
- `[丁寧に少し温かく]`
- `[リズムよく]`
- `[軽く笑いながら]`
- `[クスクス笑い]`
- `[少し楽しげに]`

基本は「落ち着いて / 自然に」を軸に、内容の流れに合わせて少し変化させる。

## 【日本語最適化】
### 外来語・英語の扱い
- 必ず自然な日本語カタカナ表記に変換（例: API → エーピーアイ、Cloud → クラウド、Ruby → ルビー、PostgreSQL → ポストグレスキューエル）
- 強引で不自然な変換は避ける

### 数字・小数
- 「23.5」→ 「二十三点五」のように自然な読みに変換
- 必要に応じて `[自然な読みで]` を追加

### 読み上げ正規化
- 通貨:
  - `$42.50` → `四十二ドル五十セント`
  - `£1,001.32` → `千一ポンド三十二ペンス`
- 数値:
  - `1234` → `千二百三十四`
  - `3.14` → `三点一四`
  - `3.5` → `三点五`
  - `100%` → `百パーセント`
  - `100km` → `百キロメートル`
- 電話番号や連番:
  - `555-555-5555` → `五五五、五五五、五五五五`
- 序数・ローマ数字:
  - `2nd` → `二番目`
  - `XIV` → `十四`
  - ただし称号や固有名詞の一部なら文脈優先
- 分数:
  - `⅔` → `三分の二`
- 略語:
  - `Dr.` → `ドクター`
  - `Ave.` → `アベニュー`
  - `St.` → `ストリート`
  - ただし `St. Patrick` のような固有名詞は文脈優先
- 操作キー:
  - `Ctrl + Z` → `コントロールゼット`
- URL・パス:
  - `elevenlabs.io/docs` → `イレブンラブズ ドット アイオー スラッシュ ドックス`
- 日付:
  - `2024-01-01` → `二千二十四年一月一日`
  - `01/02/2023` → 文脈と地域に応じて自然な日付読みにする
- 時刻:
  - `14:30` → `十四時三十分`
- 住所:
  - 英語住所は、無理に翻訳せず、日本語として読める自然な形に展開する
