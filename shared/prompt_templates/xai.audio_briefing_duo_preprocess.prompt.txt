## 【目的】
- 会話のテンポと自然さを整える
- 必要な箇所だけ xAI TTS の公式タグを使って、間・呼吸・軽い抑揚を補う
- 話者の掛け合いを壊さない

- `[lip-smack]`
- `[breath]`
- `[inhale]`
- `[exhale]`
- `[sigh]`

## 【使ってよい wrapping tags】
- `<soft>...</soft>`
- `<whisper>...</whisper>`
- `<loud>...</loud>`
- `<build-intensity>...</build-intensity>`
- `<decrease-intensity>...</decrease-intensity>`
- `<higher-pitch>...</higher-pitch>`
- `<lower-pitch>...</lower-pitch>`
- `<slow>...</slow>`
- `<fast>...</fast>`
- `<sing-song>...</sing-song>`
- `<singing>...</singing>`
- `<laugh-speak>...</laugh-speak>`
- `<emphasis>...</emphasis>`

## 【tag の使い分け】
- inline tag は、間・呼吸・短い非言語音を足したいときに使う
  - 例: `[pause]`, `[breath]`, `[chuckle]`
- wrapping tag は、語句や短い節そのものの話し方を変えたいときに使う
  - 例: `<soft>相手にやさしく返す一言</soft>`
  - 例: `<fast>少し勢いよく反応する短い節</fast>`
  - 例: `<emphasis>会話の要点</emphasis>`
- xAI TTS では wrapping tag も積極的に使ってよい
- ただし wrapping tag で包むのは、1語から短い一節までに留める
- 段落全体や発話全体を wrapping tag で包まない
- Duo では、各話者の発話の中で自然に wrapping tag を使う
- wrapping tag を使った方が自然な箇所では、inline tag だけで済ませない
- 出力全体の中で、自然に入れられるなら wrapping tag を少なくとも1回は使う

## 【wrapping tag の具体例】
- `<|speaker:0|> まずは今日の要点です。` → `<|speaker:0|> まずは<emphasis>今日の要点</emphasis>です。`
- `<|speaker:1|> そこは静かに返したいですね。` → `<|speaker:1|> <soft>そこは静かに返したい</soft>ですね。`
- `<|speaker:0|> この数字は丁寧に見たいです。` → `<|speaker:0|> この<slow>数字</slow>は丁寧に見たいです。`

---
## 【重要ルール】
- 上の公式 whitelist にあるタグだけを使う
- 新しいタグを作らない
- タグは局所的にだけ使う
- 1文に何個もタグを重ねない
- 全文をタグだらけにしない
- 語句や短い節のニュアンス調整には、可能なら wrapping tag を優先する
- wrapping tag を自然に使える箇所があるのに、inline tag だけで済ませない
- `<|speaker:0|>` と `<|speaker:1|>` は絶対に消さない
- 話者タグの順序や話者割り当てを変えない
- 各発話の意味を変えない
- 本文に新しい説明や文を足さない
- 必要な場合に限り、句読点、疑問符、感嘆符、三点リーダを軽く補ってよい

## 【日本語最適化】
- 外来語や英語は、必要なら自然な日本語カタカナ表記へ変換する（例: API → エーピーアイ、Cloud → クラウド、Ruby → ルビー、PostgreSQL → ポストグレスキューエル）
- 数字や記号は、読み上げが不自然なら自然な読みへ直す
- ただし不自然な言い換えや意味変更は避ける

### 読み上げ正規化
- 通貨:
  - `$42.50` → `四十二ドル五十セント`
  - `£1,001.32` → `千一ポンド三十二ペンス`
- 数値:
  - `1234` → `千二百三十四`
  - `3.14` → `三点一四`
  - `3.5` → `三点五`
  - `100%` → `百パーセント`
  - `100km` → `百キロメートル`
- 電話番号や連番:
  - `555-555-5555` → `五五五、五五五、五五五五`
- 序数・ローマ数字:
  - `2nd` → `二番目`
  - `XIV` → `十四`
  - ただし称号や固有名詞の一部なら文脈優先
- 分数:
  - `⅔` → `三分の二`
- 略語:
  - `Dr.` → `ドクター`
  - `Ave.` → `アベニュー`
  - `St.` → `ストリート`
  - ただし `St. Patrick` のような固有名詞は文脈優先
- 操作キー:
  - `Ctrl + Z` → `コントロールゼット`
- URL・パス:
  - `elevenlabs.io/docs` → `イレブンラブズ ドット アイオー スラッシュ ドックス`
- 日付:
  - `2024-01-01` → `二千二十四年一月一日`
  - `01/02/2023` → 文脈と地域に応じて自然な日付読みにする
- 時刻:
  - `14:30` → `十四時三十分`
- 住所:
  - 英語住所は、無理に翻訳せず、日本語として読める自然な形に展開する
