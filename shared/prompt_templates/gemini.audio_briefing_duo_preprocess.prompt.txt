## 【重要ルール】
- whitelist にあるタグだけを使う
- 新しいタグを作らない
- タグは局所的にだけ使う
- 1文に何個も入れない
- 全文をタグだらけにしない
- `<|speaker:0|>` と `<|speaker:1|>` は絶対に消さない
- 話者タグの順序や話者割り当てを変えない
- 各発話の本文は残しつつ、必要最小限のタグだけ足す

## 【日本語最適化】
- 外来語や英語は、必要なら自然な日本語カタカナ表記へ変換する（例: API → エーピーアイ、Cloud → クラウド、Ruby → ルビー、PostgreSQL → ポストグレスキューエル）
- 数字や小数は、読み上げが不自然なら自然な読みへ直す
- ただし不自然な言い換えは避ける

### 読み上げ正規化
- 通貨:
  - `$42.50` → `四十二ドル五十セント`
  - `£1,001.32` → `千一ポンド三十二ペンス`
- 数値:
  - `1234` → `千二百三十四`
  - `3.14` → `三点一四`
  - `3.5` → `三点五`
  - `100%` → `百パーセント`
  - `100km` → `百キロメートル`
- 電話番号や連番:
  - `555-555-5555` → `五五五、五五五、五五五五`
- 序数・ローマ数字:
  - `2nd` → `二番目`
  - `XIV` → `十四`
  - ただし称号や固有名詞の一部なら文脈優先
- 分数:
  - `⅔` → `三分の二`
- 略語:
  - `Dr.` → `ドクター`
  - `Ave.` → `アベニュー`
  - `St.` → `ストリート`
  - ただし `St. Patrick` のような固有名詞は文脈優先
- 操作キー:
  - `Ctrl + Z` → `コントロールゼット`
- URL・パス:
  - `elevenlabs.io/docs` → `イレブンラブズ ドット アイオー スラッシュ ドックス`
- 日付:
  - `2024-01-01` → `二千二十四年一月一日`
  - `01/02/2023` → 文脈と地域に応じて自然な日付読みにする
- 時刻:
  - `14:30` → `十四時三十分`
- 住所:
  - 英語住所は、無理に翻訳せず、日本語として読める自然な形に展開する
